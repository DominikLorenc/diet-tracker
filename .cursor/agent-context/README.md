# Agent Context

Ten folder jest używany jako stały kontekst pracy agenta w tym projekcie.

## Co tu trzymać
- skrót zasad współpracy
- checklisty przed commitem
- krótkie notatki architektoniczne
- linki do ważnych plików w `/.claude/` i `/Notes/`

## Szybkie linki
- `/.cursor/agent-context/progress.md` — aktualny stan prac i najbliższe kroki
- `/.cursor/agent-context/pre-commit-checklist.md` — checklista jakości przed commitem
- `/.cursor/agent-context/security-checklist.md` — checklista bezpieczeństwa (OWASP)

## Cel
Utrzymanie spójnego sposobu pracy w kolejnych sesjach.
